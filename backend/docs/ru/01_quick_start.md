# Быстрый старт

## Что делает этот инструмент

MIDI Cleaner принимает MIDI-файлы, сгенерированные AI, и удаляет цифровой шум, который мешает работать с ними в нотных редакторах. Представьте это как очистку черновой записи — музыка сохраняется, а артефакты удаляются.

## Начало работы

При открытии приложения вы увидите главный интерфейс:

![Главный интерфейс](/static/docs/images/01_main_upload.png)

1. **Откройте приложение** в браузере
2. **Перетащите MIDI-файл** на область загрузки или нажмите кнопку **Load MIDI**
3. **Выберите режим** в верхней панели:
   - **Preset** (по умолчанию) — выберите пресет, например «FX / Preserve» или «Auto (Recommended)»
   - **Auto Optimize** — пусть оптимизатор автоматически подберёт лучшие настройки
   - **Manual** — настройте каждый параметр вручную
4. **Нажмите «Process & Clean»**
5. **Проверьте Processing Log** (вкладка в центральной панели), чтобы увидеть, что изменилось
6. **Скачайте** очищенный MIDI-файл кнопкой **Download Clean MIDI**

Вот и всё для базового процесса. Для лучших результатов читайте далее.

## Processing Log

После каждого запуска обработки открывается вкладка **Processing Log**, показывающая:
- **Время обработки** — сколько длился процесс
- **Notes in / out** — сколько нот было на входе и на выходе
- **Пошаговая разбивка** — каждый этап очистки, что он сделал и сколько нот затронул

Вы также можете скачать эти данные в формате `report.json` кнопкой **Download Report JSON**.

## Использование Auto Optimize

Если вы хотите, чтобы инструмент автоматически подобрал оптимальные настройки:

1. Загрузите MIDI-файл
2. Нажмите кнопку **Auto Optimize** в верхней панели
3. Установите количество попыток с помощью ползунка **Max Trials** (30–40 обычно достаточно)
4. При желании включите **LLM Guidance** для AI-подсказок по параметрам
5. Нажмите **Start Optimization**
6. Дождитесь окончания (инструмент перебирает множество комбинаций параметров)
7. Нажмите **Apply Best Parameters** — настройки обновятся до оптимальных значений
8. Нажмите **Process & Clean** для генерации результата с этими настройками
9. **Скачайте** результат

## Советы

- **Начинайте с пресета**, а не с ручной настройки каждого ползунка
- **Используйте «FX / Preserve»** для большинства AI-транскрибированных стемов — это самый безопасный вариант по умолчанию
- **Проверяйте Processing Log** — если удалено слишком много нот, возможно, стоит использовать более мягкие настройки
- **Сохраняйте отчёт** — если вы нашли настройки, которые хорошо работают, отчёт зафиксирует их точно
- **Используйте History** для сравнения разных сессий обработки

## Ограничения

> **Важно:** В текущей версии поддерживается обработка только **одного трека**. Если ваш MIDI-файл содержит несколько треков с нотами, приложение покажет предупреждение и отключит обработку. Объедините треки в вашем DAW перед загрузкой или загружайте файлы с одним треком.
